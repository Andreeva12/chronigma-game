# Bug Report #002: Duplicate Tutorial Message Pop-up

**Date:** 2024-01-22
**Game:** Chronigma
**Build:** Steam public build
**Platform:** PC
**Severity:** Low
**Priority:** Low
**Status:** Open

**Summary:**
Character message from Kronos appears twice simultaneously in the second tutorial level.

**Description:**
In the level following the very first tutorial, the hint message "You can see the PowerCube on the floor, push it to the EnergyPlate to open the door and head to the next level." is displayed in two identical text boxes at the same time, overlaying each other.

**Steps to Reproduce:**
1. Complete the first tutorial level (learning time travel).
2. Enter the next level with the PowerCube and EnergyPlate.
3. Approach the trigger zone for the hint.
4. Observe the message pop-up.

**Expected Result:**
A single instance of the tutorial message should appear.

**Actual Result:**
Two instances of the identical message appear, creating a visual overlap.

**Evidence:**
- Video: `Gameplay_Videos/Log_2_Message_Kr.mp4`
- Screenshot: `Screenshots/Log_2_Message_Kr.png`

**Additional Notes:**
This is a cosmetic UI bug that does not block progression but affects user experience and polish.
